QUICK START:
------------
1. Extract: tar -xzf BNCR-GenAI-Course-Complete.tar.gz
2. Navigate: cd GenAI-to-BNCR
3. Setup: python3.11 -m venv venv && source venv/bin/activate
4. Install: pip install -r requirements.txt
5. Configure: cp .env.example .env (edit with Azure credentials)
6. Launch: jupyter lab
7. Start: Open labs/lab05/lab05_azure_openai_basics.ipynb
